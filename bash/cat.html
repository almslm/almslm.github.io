Использование команды `cat` в Bash

Команда cat (сокращение от "concatenate" — объединять) в Bash — это одна из основных и часто используемых утилит командной строки. Она позволяет просматривать содержимое файлов, создавать и объединять файлы, выполнять другие операции с текстом.

Основные функции `cat`

1. Просмотр содержимого файла

Самое простое применение cat — это вывод содержимого одного или нескольких файлов на стандартный вывод (обычно это экран вашего терминала).

cat имя_файла.txt

Если указать несколько файлов, cat выведет их содержимое последовательно:

cat файл1.txt файл2.txt

2. Создание файла

Вы можете использовать cat для создания нового файла, перенаправив ввод со стандартного ввода (клавиатуры) в файл.

cat > новый_файл.txt

После выполнения этой команды, всё, что вы будете вводить с клавиатуры, будет записываться в новый_файл.txt. Чтобы завершить ввод и сохранить файл, нажмите Ctrl+D.

3. Объединение файлов

Как следует из названия, cat отлично подходит для объединения содержимого нескольких файлов в один.

cat файл1.txt файл2.txt > объединенный_файл.txt

Эта команда возьмёт содержимое файл1.txt, затем добавит к нему содержимое файл2.txt, и всё это запишет в новый файл объединенный_файл.txt.

4. Добавление содержимого в конец файла

Используйте оператор >> для добавления содержимого в конец существующего файла, вместо того чтобы перезаписывать его.

cat новый_текст.txt >> существующий_файл.txt

Полезные опции `cat`

  • -n: Нумеровать все строки вывода.
    • cat -n файл.txt
  • -b: Нумеровать только непустые строки.
    • cat -b файл.txt
  • -s: Сжимать несколько пустых строк в одну.
    • cat -s файл.txt
  • -E: Отображать символ `$` в конце каждой строки (полезно для обнаружения скрытых символов новой строки).
    • cat -E файл.txt

Примеры использования

Пример 1: Просмотр содержимого файла конфигурации:

cat /etc/nginx/nginx.conf

Пример 2: Создание списка покупок:

cat > shopping_list.txt
Простокваша
Финики
Оливковое масло
Батон из пшеничной муки 1-го сорта
Ctrl+D

Пример 3: Объединение двух файлов логов:

cat server.log.1 server.log.2 > combined_logs.txt

Заключение

Команда cat — это мощный и гибкий инструмент для работы с текстовыми файлами в Bash. Понимание её основных функций и опций значительно упрощает работу в командной строке.

См. так же: Команда grep.
