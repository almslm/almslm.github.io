Калькулятор

Калькулятор для выполнения основных математических операций. Поддерживает как управление мышью, так и ввод с клавиатуры.

Как использовать

  • Вводите числа, нажимая на кнопки или используя клавиатуру
  • Выбирайте операцию (+, −, ×, ÷, %)
  • Нажимайте = для получения результата
  • Используйте ⌫ для удаления последнего символа
  • Кнопка C очищает все введенные данные
  • Кнопка √ вычисляет квадратный корень из текущего числа
  • Кнопка x² возводит текущее число в квадрат
  • Кнопка x³ возводит текущее число в куб
  • Кнопка ∛ извлекает кубический корень из текущего числа
  • Кнопка 📋 копирует текущее число в буфер обмена
  • Кнопка 📥 вставляет число из буфера обмена

Управление с клавиатуры

  • 0-9 — ввод цифр
  • + - * / — математические операции
  • s — вычисление квадратного корня
  • x — возведение в квадрат
  • ! — возведение в куб
  • @ — извлечение кубического корня
  • Ctrl+C — копирование числа в буфер
  • Ctrl+V — вставка числа из буфера
  • Enter или = — вычисление результата
  • Backspace — удаление последнего символа
  • Escape или C — полная очистка

Кнопка % (остаток от деления)

Кнопка % вычисляет остаток от деления первого числа на второе (операция modulo).

Примеры:

  • 10 % 3 = 1 — 10 делится на 3, остаток 1
  • 15 % 4 = 3 — 15 делится на 4, остаток 3
  • 20 % 5 = 0 — 20 делится на 5 без остатка

Кнопка √ (квадратный корень)

Кнопка √ вычисляет квадратный корень из текущего числа на экране. Это унарная операция — она применяется к числу сразу после нажатия.

Примеры:

  • √9 = 3
  • √16 = 4
  • √2 ≈ 1.4142135624

Если попытаться вычислить корень из отрицательного числа, калькулятор покажет «Ошибка».

Кнопка x² (квадрат числа)

Кнопка x² возводит текущее число в квадрат. Это унарная операция — результат появляется сразу после нажатия.

Примеры:

  • 3² = 9
  • 5² = 25
  • (−4)² = 16
  • 1.5² = 2.25

Кнопка x³ (куб числа)

Кнопка x³ возводит текущее число в куб. Это унарная операция — результат появляется сразу после нажатия.

Примеры:

  • 2³ = 8
  • 3³ = 27
  • (−2)³ = −8
  • 1.5³ = 3.375

Кнопка ∛ (кубический корень)

Кнопка ∛ извлекает кубический корень из текущего числа. В отличие от квадратного корня, кубический корень можно извлекать из отрицательных чисел.

Примеры:

  • ∛8 = 2
  • ∛27 = 3
  • ∛(−8) = −2
  • ∛2 ≈ 1.2599210499

Кнопки 📋 и 📥 (буфер обмена)

Кнопка 📋 копирует текущее число с экрана в буфер обмена. Кнопка 📥 вставляет число из буфера обмена на экран.

Как использовать:

  1. Вычислите число (например, √144 = 12)
  2. Нажмите 📋 для копирования
  3. Откройте другое приложение (блокнот, браузер и т.д.)
  4. Вставьте число через Ctrl+V или правую кнопку мыши
  5. Или скопируйте число из другого источника и нажмите 📥 для вставки в калькулятор

Также можно использовать горячие клавиши Ctrl+C для копирования и Ctrl+V для вставки.
